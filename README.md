# AccountExpress-Next-Gen
Nuevo Sistema Solo Contabilidad

## Iniciar el sistema

### Ambos servidores (recomendado):
```bash
  cd "c:\Nuevo Sistema"
  bun run dev:all
```

### Por separado:
  # Terminal 1 — Backend:
```bash
  cd "c:\Nuevo Sistema"
  bun run dev
```

  # Terminal 2 — Frontend:
```bash
  cd "c:\Nuevo Sistema"
  bun run dev:frontend
```

### Credenciales iniciales:
  Usuario: valor de ADMIN_USERNAME en tu .env
  Contraseña: valor de ADMIN_PASSWORD en tu .env
